# Quest'o'Graph
A Dalamud Plugin to explore quests and their rewards, unlocks and more.
This plugin is an rewritten and refactored planned successor to QuestMap with the same and more features.

> Quest'o'Graph's main goal is to enhance what QuestMap already did great.

### Features
- Search for quest names or their rewards (instances, tribes, minions, etc).
- See an interactive map (Graph) of quest.
- Lookup quest information
- Open quest starting locations on the map or open quests in the journal.
  
#### New
- Switch between languages
- Compact Design
- More customizability
  
#### Planned
- Integration into other Plugins
- More and Better Filter options
- Pin / Tracking Quests
- And more based on ideas and suggestions!

### How to use
1. Open Dalamud Settings 
2. Go to Experimental 
3. Add Source https://apps.irataprojects.de/ffxiv/plugins.json
4. Save and Install through Plugins
